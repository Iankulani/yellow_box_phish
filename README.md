Yellow Box Phish is a modern cyber security platform designed to support cyber security professionals, educators, researchers, red teams, blue teams, security operations centers, and cyber range environments. The platform provides a centralized command and communication framework that enables authorized users to interact with security tools and training modules through popular communication platforms such as Discord, Telegram, Slack, WhatsApp, Signal, iMessage, Google Chat, and a dedicated web application.

The primary objective of Yellow Box Phish is to simplify cyber security operations, security awareness programs, cyber defense exercises, and controlled security testing. By integrating multiple communication channels into a single ecosystem, organizations can streamline workflows, improve collaboration, and enhance their ability to respond to cyber security challenges.

# Unified Communication Platform

One of the most distinctive features of Yellow Box Phish is its multi-platform communication capability. Security teams can securely issue authorized commands, receive notifications, monitor activities, and coordinate training exercises using familiar communication tools. Instead of requiring users to switch between multiple applications, Yellow Box Phish creates a unified environment where cyber security operations can be managed efficiently.
